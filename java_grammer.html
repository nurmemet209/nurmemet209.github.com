  • 执行顺序 静态初始化函数（.method static constructor ()V）-> 实例初始化函数 （.method public constructor ()V）
  • 所有静态变量的初始化都在静态初始化函数内执行
  • 所有实例变量的初始化都在实例初始化函数（构造函数）内执行
  • 静态初始化块儿也在静态初始化函数内执行
static {
    m=6;
}
private static int m=5;

相当于

static {
    m=6;
}
private  static int m;
static{
    m=5;
}
